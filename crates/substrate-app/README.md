# substrate-app

Application layer: [`DispatchService`] (implements [`DispatchApi`]) and [`DispatchPlanner`] for deterministic engine + session-mode selection.

```toml
substrate-app = { git = "https://github.com/KooshaPari/substrate", package = "substrate-app" }
```

Included by default in the [`substrate`](../substrate) facade (`app` feature).
